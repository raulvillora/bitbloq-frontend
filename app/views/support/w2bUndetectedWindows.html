¿Están las claves que lanzan Web2board desde la web en el editor de registro?

Cuando el navegador recibe una llamada a través del protocolo web2board://, tiene que lanzar Web2board; para que funcione, se tiene que añadir una clave en el registro de Windows.
Para ver el registro de windows, pulsar las teclas Windows + R para que se abra el menu de ejecutar, y escribir "regedit. Eso nos abre el editor del registro.

Las claves que hay que revisar que están son:

  1. HKEY_CLASSES_ROOT\web2board
  2. HKEY_CLASSES_ROOT\web2board\shell
  3. HKEY_CLASSES_ROOT\web2board\shell\open
  4. HKEY_CLASSES_ROOT\web2board\shell\open\command
A la última cadena en windows de 32b ha de tener el valor:
  • C:\\Program Files\\Web2board\\web2boardLink.exe
En windows de 64b ha de tener el valor:
  • C:\\Program Files (x86)\\Web2board\\web2boardLink.exe

Si no tiene las claves en el registro, descargue y ejecute el archivo correspondiente a su versión:

  • Windows 32b: web2boardTo32.reg
  • Windows 64b: web2board.reg
